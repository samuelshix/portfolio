Wiki App

Allows users to create wikipedia posts with custom formatting through Markdown, and implements a search feature for finding entries. Creating this app helped me gain experience with coding dynamic web applications. Developed via Django.

Cryptocurrency Portfolio Tracker App

As a cryptocurrency investor, I wanted an easier way to track the performance of my investments in one place. Using what I've learned over the course of the year, I created a portfolio tracker app which prompts users for their public wallet addesses and a date, and gives users the value of their investments as well as the amount of currencies in possession. Created with HTML/CSS/JS using API's for public endpoints.
